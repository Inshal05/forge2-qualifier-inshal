# Sample Agent Loop

Use this file as the structure for the real copied Slack evidence.

## #sprint-main

Human: Plan a tiny Trello-style Kanban board with Laravel API and React UI.

Hermes: Plan posted with backend, frontend, evidence and deployment tasks.

## #agent-coder

Hermes: Build the Laravel API routes/models/migrations and React board UI.

OpenClaw: Reported completed code, local checks, remaining deployment step.

## #agent-log

Hermes autonomous run: Posted one-line progress update without a manual prompt.
